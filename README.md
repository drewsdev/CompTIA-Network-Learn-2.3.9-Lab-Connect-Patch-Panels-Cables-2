##CompTIA CertMaster Learn v9.1

In this lab, your task is to:

* Connect a patch cable between the patch panel and the Cisco switch.
* Use the following table to determine the correct ports:

| Patch Panel Port | Cisco Switch Port |
|------------------|-------------------|
| Exec 1           | Port 1            |
| Supp             | Port 6            |
| Lobby            | Port 8            |
